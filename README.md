# 📦 Professional Inventory Management System

A feature-rich **Command Line Interface (CLI) Inventory Management System** built with **Python** using **Object-Oriented Programming (OOP)** principles.  
This project helps manage products, track sales, monitor stock, generate reports, and export inventory data.

---

## 🚀 Features

### 📦 Product Management
- Add new products with unique product IDs
- View all products with inventory summary
- Update product details
- Delete products with confirmation
- Prevent duplicate product names

### 🔍 Search & Sorting
- Search products by:
  - Name
  - Category
  - Price range
  - Stock level
- Sort products by:
  - Name (A-Z)
  - Price (Low to High / High to Low)
  - Stock (Low to High / High to Low)

### 💰 Sales & Transactions
- Sell products
- Automatically reduce stock after sale
- Track sales history
- Calculate total revenue
- Show today's revenue

### 📊 Reports & Analytics
- Generate detailed inventory reports
- Category-wise product breakdown
- Low stock alerts
- Top 5 most valuable products
- Top 5 best-selling products

### 💾 Data Management
- Save inventory data in JSON format
- Save sales history in JSON format
- Automatically load saved data on startup
- Export inventory to CSV format

---

## 🛠️ Technologies Used

- **Python 3**
- **OOP (Object-Oriented Programming)**
- **JSON File Handling**
- **CSV Export**
- **Datetime Module**
- **OS Module**
